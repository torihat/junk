junk
====

junk-library for xyzzy

junk-library とは？
----

JunkScriptsで配布している拡張の共通ライブラリ的なものです。
利用条件等に関しては、LICENSE.txtをご覧下さい。

動作環境
----

xyzzy-0.2.2.235以降

インストール
----

配布アーカイブを system-root 以下に展開してください。

設定
----

* junk/http の設定は ~/.junk/config-http.l で行ってください。(config-http.l.sample参照)
